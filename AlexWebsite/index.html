Ford GoBike System Data Analysis

This project focuses on individual rides made in a bike-sharing system covering the greater San Francisco Bay area for the year 2014. The dataset had 8,081,216 rides in the dataset with 15 features initially. After cleaning the data and carring out feature engineering, the data structure went down to 7,868,282 rows of data with 19 columns. There are 2 datetime variables, 2 integer variables, 2 categorical variables, and 13 object variables.
